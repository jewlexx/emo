## Install

```bash
npm install --save emo-react
## Or with Yarn
yarn add emo-react
```

## Usage

```tsx
import React from 'react';

import Emoji from 'emo-react';

const Example = () => <Emoji emoji="🕊️" />;
```

Would return:

[![🕊️](https://twemoji.maxcdn.com/v/latest/72x72/1f54a.png)](#)

On any and every device
